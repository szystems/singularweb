# SingularWeb - NGO & Development Portfolio Website

[![HTML5](https://img.shields.io/badge/HTML5-Semantic-orange.svg)](https://developer.mozilla.org/en-US/docs/Web/HTML)
[![CSS3](https://img.shields.io/badge/CSS3-Responsive-blue.svg)](https://developer.mozilla.org/en-US/docs/Web/CSS)
[![JavaScript](https://img.shields.io/badge/JavaScript-ES6%2B-yellow.svg)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

**SingularWeb** is a multi-page responsive website built for an international development consultancy. The site showcases the organization's portfolio of projects with international NGOs and development agencies, including partnerships with organizations such as Oxfam, CARE, Rotary International, and Water for People.

## Features

- **Multi-Page Architecture** — Dedicated pages for each partner organization and project.
- **Responsive Design** — Mobile-first layout optimized for all screen sizes.
- **Portfolio Showcase** — Visual presentation of development projects and impact stories.
- **Contact System** — Integrated contact forms for inquiries and partnerships.
- **PDF Resources** — Downloadable project documentation and reports.

## Tech Stack

| Layer | Technology |
|---|---|
| **Markup** | HTML5 (Semantic) |
| **Styling** | CSS3, Bootstrap, Responsive Design |
| **Interactivity** | JavaScript |
| **Assets** | Custom fonts, optimized images |

## Project Structure
```
singularweb/
├── index.html              # Landing page
├── nosotros.html           # About page
├── portafolio.html         # Portfolio overview
├── contacto.html           # Contact page
├── [partner].html          # Individual partner/project pages
├── css/                    # Stylesheets
├── js/                     # JavaScript files
├── images/                 # Image assets
├── fonts/                  # Custom typography
└── pdf/                    # Downloadable documents
```

## Local Development

```bash
git clone https://github.com/szystems/singularweb.git
cd singularweb
# Open index.html in your browser or use a local server
npx serve .
```
